WriteQuest（離線可用）
===================
一個單檔（+manifest/sw）手機友善的寫作遊戲化小工具。資料存於瀏覽器（localStorage），可匯入/匯出 JSON。

使用方法（快速上手）
1) 將 `writequest.zip` 解壓縮。
2) 直接用手機或電腦的瀏覽器開啟 `index.html` 即可使用。
3) 若要安裝成桌面 App（PWA）：
   - 把整個資料夾上傳到任一靜態空間（例如 GitHub Pages、Netlify、Vercel 或本機簡易伺服器），
   - 用瀏覽器打開後，選單中點「加入主畫面」或「Install」。

設計規則
- 每 100 字 +10 EXP；當日 ≥150 字，再加成 +30 EXP。
- 等級公式：Lv = floor( sqrt( EXP / 50 ) ) + 1。
- 徽章：7/14/30 連續、單篇 1000 字、多文體（月內 ≥4 種）、當月 20 天。

備份
- 右上角「匯出 JSON」即可下載備份；也可「匯入」復原。
